# 🎯 Tic-Tac-Toe in Python

### Hey there, I'm Ayushi Kumari 👋

Welcome to my Tic-Tac-Toe project! This is a classic two-player console game built using Python, where players take turns placing **X** and **O** on a 3×3 board. The first player to align three symbols horizontally, vertically, or diagonally wins the game.

## 🚀 About the Project

This project was created to strengthen my understanding of:

* Python programming fundamentals
* Functions and modular code
* Conditional statements
* Loops and user input handling
* Game logic implementation

## ✨ Features

✅ Interactive command-line gameplay
✅ Two-player mode (X vs O)
✅ Input validation for incorrect moves
✅ Winner detection across rows, columns, and diagonals
✅ Draw detection when the board is full
✅ Clean and beginner-friendly code structure

## 🎮 Game Preview

```text
 1 | 2 | 3
---|---|---
 4 | 5 | 6
---|---|---
 7 | 8 | 9
```

Players choose a position from **1 to 9** and compete to create a winning combination.

## 🛠️ Tech Stack

## ▶️ Getting Started

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:

   ```bash
   cd tic-tac-toe-python
   ```

3. Run the game:

   ```bash
   python tic_tac_toe.py
   ```
